What does this app do?

The Ounces Optimizer helps you decide which stopes and development headings to haul ore from - and how many trucks to assign to each - to hit your daily ounces target.

How ounces are calculated

Ounces = Tonnes x Grade x Ounces Factor

Where:

  • Tonnes = Trucks x Truck Factor (51 t/truck)
  • Grade = g/t of the stope or development heading
  • Ounces Factor = 0.03215
  • Development: 1 cut = 350 t | Default grade = 5.7 g/t

How to use

  • Set Ounces Target, Plan Tonnes, Ounces Factor
  • Add ore stopes with Stock and Grade
  • Add development headings by entering number of cuts (1 cut = 350t)
  • Edit any stope or heading inline using the pencil button
  • Toggle ON/OFF and set Cap to limit trucks per source
  • Tap Find Best 3
